# Development

## Installation

Prerequisites:
- [Node](https://nodejs.org/) >= 14
- [pnPm](https://pnpm.io/installation) 6.x

First, install all the dependencies.
```sh
pnpm install
```

Finally start the development server.
```sh
npm start
```

## Lint

[ESLint](https://eslint.org/) is used to check the code.

Run the linter (ESLint):
```sh
npm run lint
```
